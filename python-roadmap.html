Learning Roadmap

Master Python
from Zero

A structured, phase-by-phase guide that takes you from writing your first line of Python to building real-world projects with confidence.

Phase 1 — Foundations Phase 2 — Core Python Phase 3 — Intermediate Phase 4 — Libraries Phase 5 — Advanced Phase 6 — Specialise Resources

How to use this roadmap: Work through each phase in order. Don't rush — understanding is more important than speed. Build at least one small project per phase to solidify what you learn.

1

Foundations Get Python running and learn the absolute basics

Setup & Environment

  • Install Python 3 (python.org)
  • Choose an editor — VS Code or PyCharm
  • Run your first script: print("Hello, World!")
  • Use the interactive REPL (python3)
  • Understand pip for packages

Variables & Data Types

  • int, float, str, bool
  • Variable assignment and naming rules
  • type() and dynamic typing
  • None — the null value
  • Type casting: int(), str(), float()

Operators

  • Arithmetic: + - * / // % **
  • Comparison: == != < > <= >=
  • Logical: and or not
  • Assignment operators: += -= *=
  • Membership: in, not in

Input & Output

  • print() — output to the console
  • input() — read user input
  • f-strings for formatted output
  • format() and % formatting

Control Flow

  • if / elif / else
  • Indentation rules (Python uses whitespace)
  • Nested conditions
  • Ternary expression: x if cond else y

Loops

  • for loop and range()
  • while loop
  • break, continue, pass
  • else clause on loops
  • Nested loops

2

Core Python Data structures, functions, and error handling

Lists & Tuples

  • Create, index, and slice lists
  • List methods: append, pop, sort, reverse
  • Tuples — immutable sequences
  • Unpacking: a, b = (1, 2)
  • Nested lists (2D arrays)

Dictionaries & Sets

  • Create and access dict keys/values
  • Dict methods: get, keys, values, items
  • Sets — unique, unordered elements
  • Set operations: union, intersection, difference
  • defaultdict from collections

Strings in Depth

  • Slicing, indexing, and iteration
  • String methods: split, join, strip, replace
  • upper, lower, title, find
  • String immutability
  • Multi-line strings with triple quotes

Functions

  • Define with def, call, and return
  • Positional and keyword arguments
  • Default parameter values
  • *args and **kwargs
  • Scope: local vs global variables
  • Lambda (anonymous) functions

Error Handling

  • try / except / else / finally
  • Common exceptions: ValueError, TypeError, KeyError
  • Raise custom exceptions with raise
  • Define custom exception classes

File I/O

  • Open, read, write files with open()
  • Modes: r, w, a, rb
  • The with statement (context manager)
  • Reading CSV files with csv module
  • Working with JSON: json.load / json.dump
Phase 3 starts here
3

Intermediate Python OOP, comprehensions, modules, and the standard library

Object-Oriented Programming

  • Classes and objects
  • __init__, self, and instance attributes
  • Instance methods, class methods, static methods
  • Inheritance and method overriding
  • Encapsulation: private/protected attributes
  • Polymorphism and duck typing
  • Dunder (magic) methods: __str__, __repr__, __len__

Comprehensions

  • List comprehensions: [x*2 for x in lst]
  • Dict comprehensions: {k: v for k, v in d.items()}
  • Set comprehensions
  • Generator expressions: (x for x in range(1000))
  • Conditional filtering in comprehensions

Modules & Packages

  • Import a module: import os
  • From-import: from math import sqrt
  • Create your own module
  • Packages: __init__.py
  • Virtual environments with venv

Useful Standard Library

  • os and pathlib — file system
  • sys — interpreter interaction
  • datetime — dates and times
  • re — regular expressions
  • collections — namedtuple, Counter, deque
  • itertools and functools
  • random and math

Testing Basics

  • Write unit tests with unittest
  • assert statements
  • pytest — simpler test framework
  • Test structure: arrange, act, assert
  • Mocking with unittest.mock

Iterators & Generators

  • iter() and next()
  • Create a custom iterator class
  • Generator functions with yield
  • Benefits: lazy evaluation, memory efficiency
  • yield from for delegation

4

Key Libraries The ecosystem that makes Python powerful

NumPy

  • Arrays vs Python lists
  • Array creation: arange, zeros, linspace
  • Slicing, indexing, reshaping
  • Vectorized operations (no loops)
  • Mathematical functions: mean, std, dot

Pandas

  • Series and DataFrame
  • Read CSV / Excel: pd.read_csv()
  • Filtering, selection (loc, iloc)
  • GroupBy and aggregations
  • Handling missing data (NaN)
  • Merging and joining DataFrames

Matplotlib & Seaborn

  • Line, bar, scatter, and histogram plots
  • Labels, titles, and legends
  • Subplots and figure sizing
  • Seaborn for statistical plots
  • Save figures to file

Requests & APIs

  • HTTP basics: GET, POST, headers
  • requests.get(url) and .json()
  • Handling response status codes
  • Authentication (API keys, OAuth)
  • Consuming public REST APIs

Automation with Python

  • Automate file operations with os / shutil
  • Schedule tasks with schedule library
  • Send emails with smtplib
  • Parse web pages with BeautifulSoup
  • Automate Excel with openpyxl

Databases

  • SQLite with built-in sqlite3
  • CRUD operations (Create, Read, Update, Delete)
  • ORM basics with SQLAlchemy
  • Connect to PostgreSQL / MySQL
  • Use Pandas to query a database

5

Advanced Python Deep language features and performance patterns

Decorators

  • Functions as first-class objects
  • Higher-order functions
  • Write a simple decorator
  • @wraps from functools
  • Parametrized decorators
  • Class-based decorators

Concurrency & Async

  • Threads with threading module
  • Processes with multiprocessing
  • asyncio — event loop basics
  • async def and await
  • aiohttp for async HTTP
  • When to use threads vs processes vs async

Type Hints & Annotations

  • Basic annotations: def f(x: int) -> str
  • List, Dict, Optional from typing
  • Static analysis with mypy
  • Dataclasses with typed fields
  • Pydantic for data validation

Performance & Profiling

  • Profile with cProfile and timeit
  • Big-O complexity awareness
  • Memory optimization techniques
  • __slots__ for memory-efficient classes
  • Speed up with NumPy and vectorization

Context Managers

  • __enter__ and __exit__ protocol
  • Write a custom context manager
  • @contextmanager from contextlib
  • Use cases: files, locks, DB connections

Design Patterns in Python

  • Singleton, Factory, Observer patterns
  • Strategy pattern with functions
  • Dependency injection
  • Repository pattern for data access

6

Specialise Choose a domain and go deep

Web Development

  • Flask — lightweight micro-framework
  • Django — full-stack web framework
  • FastAPI — modern async REST APIs
  • Templates, routing, middleware
  • Authentication and sessions

Data Science & ML

  • Exploratory Data Analysis (EDA)
  • scikit-learn — ML algorithms
  • Supervised vs unsupervised learning
  • Deep learning with TensorFlow / PyTorch
  • Jupyter Notebooks for exploration

DevOps & Automation

  • Ansible for configuration management
  • CI/CD pipeline scripts
  • Docker SDK for Python
  • Kubernetes client library
  • Infrastructure as Code with Pulumi

Security / Pentesting

  • Network scanning with scapy
  • Hashing and cryptography: hashlib, cryptography
  • Build simple port scanners
  • Automate security checks
  • Capture and parse network packets

Telecom & RF with Python

  • Parse drive-test log files
  • Compute KPIs with Pandas
  • Plot coverage maps with Matplotlib / Folium
  • Automate post-processing workflows
  • Integrate with TEMS / Actix data exports

Best Practices & Packaging

  • PEP 8 code style + black formatter
  • Linting with flake8 / ruff
  • Semantic versioning
  • Package with pyproject.toml
  • Publish to PyPI
  • Write clear documentation with docstrings
