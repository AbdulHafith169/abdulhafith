# Abdul Hafeez Mohamed Abu-Grara — Portfolio (GitHub Pages)

This is a simple GitHub Pages friendly portfolio built with:
- HTML
- CSS
- Small JavaScript (theme toggle, mobile nav, scroll animations)

## How to use (GitHub Pages)
1. Create a repository (example: `abdulhafith169.github.io` or `portfolio`)
2. Upload these files to the repository root:
   - `index.html`
   - `style.css`
   - `script.js`
3. In GitHub:
   - **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** (or `master`) and **/(root)**

## Customize
- Replace:
  - `your.email@example.com`
  - LinkedIn URL
  - GitHub URL
  - Add your CV PDF and update the "Download CV" link in `index.html`

Example:
`href="./assets/Abdul-Hafeez-CV.pdf"`

(If you add that file, create an `assets/` folder.)
